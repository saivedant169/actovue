# Costs

Every GPU session and annotation batch gets a line here, so the total is never a
surprise and the published numbers can be tied to the hardware that produced them.
Local CPU development on the laptop costs nothing and is not tracked.

| GPU: probe training, capture, benchmark (roughly 150 to 250 H100-hours) | 400 to 600 USD |
| Annotation API (frontier model plus web search for entity labels) | 200 to 300 USD |

Iterate on a cheap card (A100, or a local RTX) and reserve H100-class time for the
numbers that get published, since throughput is not comparable across cards.
